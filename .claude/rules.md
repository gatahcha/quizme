# QuizMe Project Rules

## Quiz and Mindmap Generation

When generating quizzes or mindmaps from markdown content:

- **Mindmap generation rules**: Follow the detailed specifications in `@prompts/cli-mindmap-generator.md`
- **Quiz generation rules**: Follow the detailed specifications in `@prompts/cli-quiz-generator.md`

These prompt files contain the complete schemas, conversion rules, and output requirements for generating properly formatted JSON files from structured educational markdown content.
